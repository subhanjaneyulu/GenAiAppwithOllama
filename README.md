# GenAiAppwithOllama
A simple Generative AI application built using:

- Streamlit
- LangChain
- Ollama
- Gemma 2B

## Description
A simple Generative AI chatbot built using Streamlit, LangChain and Ollama.

## Features
- User question input
- Local LLM inference using Ollama
- Prompt Templates with LangChain
- Output Parsing

## Run

pip install -r requirement.txt

streamlit run app.py
